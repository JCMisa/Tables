Markup Tag Tag Description Possible Attributes Attributes Description
<a></a> defines a hyperlink as a link that points to another website or material. > href
> target		 > Specifies the URL of the linked resource.
> Specifies where to open the linked document (same tab or new tab)
<img> Inserts an image into the document. > src
> alt
> width
> height		 > Specifies the URL of the image.
> Provides alternative text for the image.
> Specifies the width of the image.
> Specifies the height of the image.
<div> Defines a division or section in an HTML document > class
> id		 > Usually used as identifier for the tag when applying styles or logic.
> Specifies a unique identifier for the element.
<p> Defines a paragraph of text > class
> id		 > Usually used as identifier for the tag when applying styles or logic.
> Specifies a unique identifier for the element.
<h1> to <h6> Define headings of different levels, from most important (<h1>) to least important (<h6>). > class
> id		 > Usually used as identifier for the tag when applying styles or logic.
> Specifies a unique identifier for the element.
<span> Defines a generic inline container used to group elements for styling purposes. > class
> id		 > Usually used as identifier for the tag when applying styles or logic.
> Specifies a unique identifier for the element.
<input> Defines an input control that allows user to enter data > type
> name
> value		 > Specifies the type of input control (text, checkbox, radio)
> Specifies the name of the input field
> Specifies the initial value of the input field.
<button> Defines a clickable button > type
> name
> value		 > Specifies the type of button (submit, reset, button)
> Specifies the name of the button.
> Specifies the value associated with the button.
<ul> Defines an unordered list > class
> id		 > Usually used as identifier for the tag when applying styles or logic.
> Specifies a unique identifier for the element.
<ol> Defines an ordered list > class
> id		 > Usually used as identifier for the tag when applying styles or logic.
> Specifies a unique identifier for the element.
<li> Defines a list item within an ordered or unordered list. > class
> id		 > Usually used as identifier for the tag when applying styles or logic.
> Specifies a unique identifier for the element.
<table> Defines a table for organizing tabular data. > border
> width
> cellspacing		 > Specifies the width of the border around the table.
> Specifies the width of the table.
> Specifies the space between cells.
<tr> Defines a row within a table > class
> id		 > Usually used as identifier for the tag when applying styles or logic.
> Specifies a unique identifier for the element.
<td> Defines a data cell within a table row. > colspan
> rowspan		 > Specifies the number of columns the cell should span.
> Specifies the number of rows the cell should span.
<form> Defines an HTML form for user input > action
> method		 > Specifies the URL where the form data should be submitted.
> Specifies the HTTP method used to submit the form data (GET, POST).
<textarea> Defines a multi-line text input control. > name
> rows
> cols		 > Specifies the name of the textarea.
> Specifies the number of visible rows in the textarea (width).
> Specifies the number of visible columns in the textarea (height)
<select> Defines a dropdown list > name
> multiple		 > Specifies the name of the select element.
> Specifies whether multiple options can be selected.
<option> Defines an option within a dropdown list > value
> selected		 > Specifies the value associated with the option.
> Specifies that the option should be pre-selected.
<label> Defines a label for an input element > for > Specifies the id of the input element the label is associated with.
<iframe> Defines an inline frame, which allows another HTML document to be embedded within the current document. > src
> width, height		 > Specifies the URL of the document to be embedded.
> Specifies the dimensions of the iframe.
